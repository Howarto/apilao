# APILao - AGPL Fork of Bruno

An open-source API client built on Bruno, converted to AGPL to ensure all derivative works remain free and open source.

## About This Fork

**APILao** is a fork of [Bruno](https://github.com/usebruno/bruno), the innovative API client by the Bruno team. This fork was created with the intention to convert the project to **AGPL-3.0**, ensuring that:

- All features and improvements remain free and open source
- Future derivative works cannot be privatized or hidden behind paywalls
- The spirit of open-source collaboration is preserved

### Why This Fork?

The Bruno team has progressively moved many valuable features behind their commercial versions, limiting access to functionality that many users depend on. This fork commits to keeping all features accessible to everyone under the AGPL license.

### Respecting Original License

While this fork is licensed under **AGPL-3.0**, we fully respect the original **MIT license** under which Bruno was created.

- The original MIT license is preserved as `LICENSE-ORIGINAL` and remains applicable to files and code from the original Bruno project, as documented in the Git commit history.
- All files and their attribution to the original Bruno project can be tracked through Git commit history
- Contributors to the original Bruno project are acknowledged and credited

For more information about these licenses:
- [AGPL-3.0 License](https://www.gnu.org/licenses/agpl-3.0.en.html)
- [MIT License](LICENSE-ORIGINAL)
